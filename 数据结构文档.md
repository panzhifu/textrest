## 4. 数据库结构设计

### 4.1 数据库表结构

| 表名 | 说明 |
|------|------|
| books | 存储书籍信息 |
| chapters | 存储章节信息 |
| book_sources | 存储书源信息 |
| read_records | 存储阅读记录 |
| configs | 存储通用配置 |

### 4.2 表结构详情

#### 4.2.1 books 表

| 字段名 | 类型 | 约束 | 描述 | 索引 |
|-------|------|------|------|------|
| book_id | TEXT | PRIMARY KEY | 书籍唯一标识(网络书籍使用url, 本地书籍使用id) | PRIMARY |
| name | TEXT | NOT NULL | 书名 | INDEX |
| author | TEXT | NOT NULL | 作者 | INDEX |
| kind | TEXT | | 分类 | INDEX |
| cover_url | TEXT | | 封面URL（网络图片URL或本地文件路径） | |
| intro | TEXT | | 简介 | |
| origin | TEXT | | 书源URL | INDEX |
| book_type | TEXT | NOT NULL | 书籍类型 | INDEX |
| word_count | INTEGER | DEFAULT 0 | 字数 | |
| latest_chapter_title | TEXT | | 最新章节标题 | |
| toc_url | TEXT | | 目录页URL （网络目录页URL或本地文件路径） | |
| book_group | TEXT | | 书籍分组 | INDEX |
| add_time | DATETIME | NOT NULL DEFAULT CURRENT_TIMESTAMP | 添加时间 | INDEX |
| last_read_time | DATETIME | | 最后阅读时间 | INDEX |
| isbn | TEXT | | ISBN | |
| publisher | TEXT | | 出版社 | |
| publish_date | TEXT | | 出版日期 | |
| status | TEXT | DEFAULT '未知' | 书籍状态 | |

#### 4.2.2 chapters 表

| 字段名 | 类型 | 约束 | 描述 | 索引 |
|-------|------|------|------|------|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 章节ID | PRIMARY |
| book_id | TEXT | NOT NULL, FOREIGN KEY REFERENCES books(book_id) | 所属书籍ID | INDEX |
| title | TEXT | NOT NULL | 章节标题 | |
| url | TEXT | NOT NULL | 章节URL（网络URL或本地路径） | |
| chapter_index | INTEGER | NOT NULL | 章节索引 | INDEX |
| content | TEXT | | 章节内容 | |
| word_count | INTEGER | DEFAULT 0 | 章节字数 | |
| is_vip | BOOLEAN | DEFAULT FALSE | 是否VIP章节 | |

#### 4.2.4 read_records 表

| 字段名 | 类型 | 约束 | 描述 | 索引 |

| book_id | TEXT | PRIMARY KEY, FOREIGN KEY REFERENCES books(book_id) | 书籍ID | PRIMARY |

| dur_chapter_pos | INTEGER | DEFAULT 0 | 当前章节位置 | |

| last_chapter_pos | INTEGER | DEFAULT 0 | 最后章节位置 | |
| total_read_time | INTEGER | DEFAULT 0 | 总阅读时长（秒） | |


#### 4.2.5 configs 表

| 字段名 | 类型 | 约束 | 描述 | 索引 |
|-------|------|------|------|------|
| config_key | TEXT | PRIMARY KEY | 配置键 | PRIMARY |
| config_value | TEXT | NOT NULL | 配置值（JSON格式） | |
| config_type | TEXT | NOT NULL DEFAULT 'general' | 配置类型（frontend/backend/general） | INDEX |
| update_time | DATETIME | NOT NULL DEFAULT CURRENT_TIMESTAMP | 更新时间 | |
| version | INTEGER | DEFAULT 1 | 版本号 | |
